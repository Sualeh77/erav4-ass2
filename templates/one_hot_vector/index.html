About One-Hot Encoding

What is One-Hot Encoding?

One-hot encoding is a way to represent categorical data (like words) as binary vectors:

  • Each word gets a unique position in the vector
  • The vector is filled with zeros
  • Only one position is set to 1 (hence "one-hot")
  • Vector length = vocabulary size
Example:

For vocabulary: ["cat", "dog", "fish"]

  • cat → [1, 0, 0]
  • dog → [0, 1, 0]
  • fish → [0, 0, 1]
Use Cases: Natural Language Processing, Machine Learning feature encoding, Neural Network inputs
